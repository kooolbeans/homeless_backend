# Homeless Backend

## Requirements
- mongodb v. >= 4
- nodejs v. >= 18
- npm v. >= 8

## Install
- npm install
- copy env.example to .env

## Run server
- npm start

## Run tests
- npm run test

## Endpoints

### GET /v1/api/homies Retrives all
### GET /v1/api/homie/:id Retrive by id
### POST /v1/api/homie Insert Profile
### PUT /v1/api/homie/:id Update Profile
### DELETE /v1/api/homie/:id Delete by id
